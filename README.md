# Tuckshop Inventory System
CSI142 Group Mini-Project | Semester II 2025/26 | Group 35

## Project Overview
A Java console application that helps a Botswana tuckshop owner manage 
inventory and track sales. Tuckshop owners in Botswana often struggle to 
track stock and know if they are making a profit or loss. This system 
solves that by allowing the owner to add items, monitor stock levels, 
record sales, and view a sales summary.

## How to Compile and Run
1. Clone the repository :git clone https://github.com/atlangkeolebile-svg/tuckshop-inventory-system.git
2. Navigate to the `src` folder:cd tuckshop-inventory-system/src
3. Compile:javac model/.java service/.java app/*.java
4. Run:java app.Main
5. ## Features
1. Add Item — add a new product with name, price and quantity
2. View Inventory — display all current stock
3. Update Stock — restock an existing item
4. Record Sale — sell an item and reduce stock
5. View Sales Summary — see all sales and total revenue
6. Exit

## Domain Context
Tuckshops are small neighbourhood shops common across Botswana. 
Owners rarely have tools to track stock or measure profit, making 
this system a practical and locally relevant solution.
